🧪 Testes Unitários

Esta página executa testes unitários automatizados das funções principais do EcoAlerta. Os testes garantem que a lógica de cálculo de próxima coleta, reportes e outras funcionalidades estão corretas.

📋 O que está sendo testado

  • parseDiaSemana: Conversão de nomes de dias para números
  • parseDiaHora: Extração de dia e horário de strings
  • nextCollection: Cálculo da próxima coleta (casos diversos)
  • isVesperaColeta: Detecção de véspera de coleta
  • addReport: Geração de protocolos e salvamento
  • formatHorasRestantes: Formatação de tempo restante
  • saveConfig/loadConfig: Persistência de configuração
Carregando testes...

💡 Como funciona

Os testes usam um mini framework desenvolvido especialmente para este projeto, sem dependências externas. Cada teste valida um comportamento específico das funções utilitárias em app.js.

Testes que passam aparecem com ✅, e testes que falham com ❌. Se todos os testes passarem, o sistema está funcionando corretamente!
